# Sistema de Gestión de Inventario

Aplicación web para la gestión y control de inventario de productos, desarrollada como proyecto full stack utilizando Java, Spring Boot y Angular.

El sistema permite administrar productos, controlar existencias y gestionar información de inventario de forma eficiente mediante una interfaz web moderna.

---

## Tecnologías utilizadas

Backend
- Java
- Spring Boot
- Spring Data JPA
- API REST

Frontend
- Angular
- TypeScript
- HTML5
- CSS

Base de datos
- MySQL / SQL

Control de versiones
- Git
- GitHub

---

## Funcionalidades principales

- Registro y gestión de productos
- Crear, editar y eliminar productos
- Consulta de inventario
- Gestión de stock
- API REST para comunicación entre frontend y backend
- Interfaz web responsive

---

## Arquitectura del proyecto

El proyecto está dividido en dos partes principales:

Frontend
Aplicación desarrollada en Angular encargada de la interfaz de usuario y consumo de la API.

Backend
API REST desarrollada en Java con Spring Boot encargada de la lógica de negocio y acceso a la base de datos.
